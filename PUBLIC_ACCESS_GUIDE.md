# Public Mobile Access

Use this when employees are not on the same Wi-Fi.

## One-click start

Double-click:

`START_PUBLIC_LINK.bat`

It starts Yaser Mall Stock and then tries to create a public HTTPS tunnel.

## If Cloudflare Tunnel is installed

The window will show a link like:

`https://something.trycloudflare.com`

Send employees:

- Employee: `https://something.trycloudflare.com/employee`
- Admin: `https://something.trycloudflare.com/admin/items`
- Dashboard: `https://something.trycloudflare.com/dashboard`

## If ngrok is installed

The window will show a link like:

`https://something.ngrok-free.app`

Send employees:

- Employee: `https://something.ngrok-free.app/employee`
- Admin: `https://something.ngrok-free.app/admin/items`
- Dashboard: `https://something.ngrok-free.app/dashboard`

## Install on phone

Android Chrome:

1. Open the public Employee link.
2. Tap menu.
3. Tap Add to Home screen.

IPhone Safari:

1. Open the public Employee link.
2. Tap Share.
3. Tap Add to Home Screen.

## Important

Keep the computer and tunnel window open while employees are using the app.
